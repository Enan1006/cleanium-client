# Cleanium
A full-stack MERN web application <br/>
Live Link:
https://cleanium-e9798.web.app/

## Features
- Dark/ Light theme
- Admin/Registered user authorization
- Only admin can't access dashboard route
- Hire cleaners
- User removal from admin panel dashboard

## Technologies
- React Query
- Axios
- React Router
- Firebase
- AOS
